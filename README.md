> 此模版是基于layui前端UI框架，风格简单、舒适。

依赖模块
===
- `layui`
- `jquery-1.12.1.min.js`
- `underscore-min.js`

启动说明
===
```
// 安装开发环境
$ npm i -g fis3

// 进入项目根目录
$ cd **

// 启动服务
$ npm run server

// 开发调试
$ npm run dev

// 发布项目
$ npm run pro
```
